Default contracts

There are several system contracts that are deployed in the root namespace on all Kadena networks and chains. These contracts provide the basic framework for working with tokens and accounts in a digital ledger. You can explore the functions and capabilities that are defined in the contracts directly in the source code or by using the Developer Tools Module Explorer.
